# ai_content_app
ai工具，发布文章


# 项目介绍
希望开发一个AI工具，能够根据用户输入的文章主题，自动生成符合要求的文章。
并能够根据用户输入的文章要求，自动调整文章的长度、风格、格式等。
分发：
希望能够将该工具分发到不同的平台，包括但不限于：
- 个人博客
- 社交媒体（如Twitter、Facebook等）
- 内容管理系统（如WordPress、Jekyll等）
- 其他自定义平台

## 开发环境
- 操作系统：Windows 11
- 开发工具：PyCharm
- 编程语言：Python 3.10
- 框架：FastAPI
- 数据库：SQLite
- 其他依赖：
  - transformers
  - torch
  - uvicorn
  - fastapi-utils
